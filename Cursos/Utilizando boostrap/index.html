Sobre mim

Profissional formado em engenharia civil com desenvolvimento amplo da capacidade de resolver problemas! Paixão pelo desenvolvimento de sistemas e soluções que melhorem o cotidiano das pessoas e trazem melhorias para aqueles com quem tenho o prazer de trabalhar. Durante minha formação sempre possuí grande interesse na área de desenvolvimento de interfaces e softwares, o que me levou a produzir uma aplicação web (utilizando javascript e bootstrap) para cálculos estruturais de concreto armado. Atualmente possuo conhecimento para atuação front-end com HTML, CSS, Jquery, Bootstrap, Vue.js e Javascript mas também para atuação back-end com desenvolvimento e testes em Python e APIs com integração a banco de dados como o MongoDB!
